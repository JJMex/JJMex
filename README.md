# 🇲🇽 JJMex Community Hub

Bienvenido al repositorio central de **JJMex**, una infraestructura digital diseñada para la orquestación de datos en tiempo real, monitoreo de movilidad y desarrollo de potencial humano. 

Este proyecto integra una interfaz web de alto nivel con una flota de automatizaciones y comunidades gestionadas por un núcleo de ejecución central.

---

## 📋 Resumen del Ecosistema

El ecosistema JJMex opera bajo una arquitectura colaborativa y automatizada, dividida en tres pilares fundamentales:

### 1. 🌐 Elite Web Interface
Un centro de mando minimalista y profesional construido con una estética *Glassmorphism* que sirve como punto de acceso único para la comunidad.
* **Diseño Dinámico:** Interfaz inmersiva con iluminación ambiental y texturas de grado cinemático.
* **Generación Autónoma:** Creación de códigos QR On-Demand para cada comunidad sin necesidad de imágenes estáticas.
* **UX/UI:** Identificación visual por plataforma (Telegram/WhatsApp) con micro-interacciones de alta respuesta.

### 2. 📡 Red de Monitoreo Inteligente
Sistemas de recolección de datos y reportes comunitarios enfocados en la eficiencia operativa diaria:
* **🚇 Alertas metro CDMX:** Vigilancia conjunta y reportes en tiempo real del estatus de las 12 líneas del sistema de transporte masivo.
* **🛣️ Alertas viales:** Monitoreo táctico del trayecto Acapulco - Cuernavaca - CDMX para la prevención de incidentes y gestión de tráfico.
* **🐺 Mente & Exito 365:** Canal de distribución de estrategias de alto rendimiento, estoicismo y crecimiento personal.

### 3. 🤖 JJMex_Bot (Master Orchestrator)
El cerebro de la operación. Un bot central encargado de ejecutar, sincronizar y supervisar la integridad de todos los procesos autónomos del ecosistema.
* **Orquestación:** Gestiona y dispara los flujos de trabajo de los distintos repositorios y servicios de la red.
* **Sincronización:** Asegura latencia cero en la transmisión de información hacia los canales oficiales de Telegram.

## 🛠️ Tecnologías y Estructura

El frontend está optimizado para velocidad, seguridad y accesibilidad, sin depender de frameworks pesados:

* **Estructura:** HTML5 Semántico.
* **Estilos:** CSS3 nativo (Variables CSS, Backdrop-filter, Animaciones Keyframe, CSS Grid/Flexbox).
* **Interactividad:** JavaScript Vanilla (Lógica de QRs y gestión de Modales).
* **Librerías Externas:** `qrcode.js` para renderizado en cliente.
* **Fuente:** *Outfit* (Google Fonts).
